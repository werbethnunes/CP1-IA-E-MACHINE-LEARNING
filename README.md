# Checkpoint 01 - Machine Learning

Aplicação do pipeline de Machine Learning utilizando um dataset de vendas de cafeteria.

O objetivo do projeto foi analisar os dados e treinar modelos capazes de identificar padrões nas vendas.

## Dataset

Dataset utilizado: **Global Coffee Shop Sales Dataset**  
Disponível no Kaggle.

A base contém informações como produto, cidade, tipo de loja, preço, quantidade e valor total das vendas.

## O que foi feito

- Preparação dos dados (tratamento de valores nulos e transformação de variáveis)
- Criação de uma variável de classificação (venda alta ou baixa)
- Divisão dos dados em treino e teste
- Treinamento de um modelo de classificação (KNN)
- Treinamento de um modelo de regressão (Regressão Linear)
- Avaliação dos modelos com métricas e gráficos

## Resultados

No modelo de classificação, foi obtida uma acurácia de aproximadamente 88%, indicando um bom desempenho.

No modelo de regressão, o valor de R² foi de aproximadamente 0.85, mostrando que o modelo conseguiu prever bem os valores das vendas.
